## ğŸš€ Step-by-Step: Submit to Google

### 1. Verify Your Site in Google Search Console

1. Go to [Google Search Console](https://search.google.com/search-console)
2. Click "Add Property"
3. Enter your domain: `https://www.buildmyhabitat.com`
4. Choose verification method:
   - **Recommended**: HTML file upload
   - **Alternative**: HTML tag (add to `app/layout.js` head)
   - **Easiest**: Domain name provider (if using Vercel, they can help)

### 2. Submit Your Sitemap

Once verified:
1. In Search Console, go to "Sitemaps" in the left menu
2. Enter: `https://www.buildmyhabitat.com/sitemap.xml`
3. Click "Submit"
4. Google will start crawling your site (can take a few days)

### 3. Request Indexing for Key Pages

After submitting sitemap, manually request indexing for important pages:
1. Use the URL Inspection tool in Search Console
2. Enter each URL and click "Request Indexing":
   - `https://www.buildmyhabitat.com`
   - `https://www.buildmyhabitat.com/build/betta`
   - `https://www.buildmyhabitat.com/build/leopard-gecko`
   - `https://www.buildmyhabitat.com/browse`
   - `https://www.buildmyhabitat.com/research`

### 4. Monitor Your Progress

- **Coverage Report**: Check which pages are indexed
- **Performance Report**: See search queries and clicks
- **Enhancements**: Verify structured data is recognized
